PostActionBuilder 

class PostActionBuilder extends ActionBuilder[PostRequest, AnyContent] with RequestMarkerContext with HttpVerbs

The action builder for the Post resource.

This is the place to put logging, metrics, to augment the request with contextual data, and manipulate the result.
